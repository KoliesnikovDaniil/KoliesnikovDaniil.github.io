Волонтерство поруч з вами

Ця сторінка демонструє використання JavaScript для роботи з DOM, подіями, циклами та умовами. Список ініціатив формується динамічно з масиву даних.

VolunteerHub допомагає координувати людей, які хочуть підтримувати громаду: допомагати літнім людям, проводити екологічні акції, збирати речі для нужденних і підтримувати притулки.

Актуальні можливості

Доступні ініціативи

Відображаються лише ініціативи, які ще не завершилися. Для цього у JavaScript використано цикл while та умовну перевірку дати.

Про нас

Ми об’єднуємо волонтерів, громадські організації та місцеві ініціативи, щоб допомога була швидкою, прозорою та зручною для кожного учасника.

Наша місія — розвивати культуру взаємодопомоги та підтримувати корисні соціальні проєкти.
