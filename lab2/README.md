## Опис реалізації

У межах лабораторної роботи було створено інтерактивну веб-сторінку платформи VolunteerHub для організації волонтерських ініціатив. Сторінка продовжує тематику першої лабораторної роботи, але основний акцент зроблено на використанні JavaScript.

У файлі `index.html` реалізовано структуру сторінки з розділами: «Доступні ініціативи», «Мої ініціативи», «Про нас» та формою «Запропонувати ініціативу». До HTML-файлу підключено зовнішній JavaScript-файл `script.js` перед закриваючим тегом `</body>`, щоб усі елементи сторінки були доступні для роботи з DOM.

У файлі `styles.css` описано зовнішній вигляд сторінки: кольори, шрифти, відступи, сітку карток, кнопки, футер, адаптивність та ефекти наведення. Для розділу з ініціативами використано CSS Grid, щоб картки коректно розташовувалися на різних екранах.

У файлі `script.js` реалізовано основну інтерактивність сторінки. Дані про волонтерські ініціативи зберігаються у масиві об’єктів. Кожна ініціатива має назву, дату, місце проведення, кількість потрібних волонтерів та короткий опис.

Для виконання вимог варіанту використано цикл `while`, який проходить по масиву ініціатив і відображає тільки ті проєкти, дата яких ще не минула. Завершені ініціативи не потрапляють у розділ «Доступні ініціативи».

Для кожної актуальної ініціативи динамічно створюється HTML-картка за допомогою `document.createElement()` та додавання вмісту через `innerHTML`. Картки додаються до сторінки через DOM-метод `appendChild()`.

На кожній картці є кнопка «Приєднатися». При натисканні на неї виконується обробник події `addEventListener('click', ...)`. Ініціатива додається до розділу «Мої ініціативи», текст кнопки змінюється на «Ви приєдналися», кнопка блокується, а кількість потрібних волонтерів зменшується на 1. Для цього використано умовну логіку `if-else`.

Також реалізовано додаткові інтерактивні елементи: кнопка показу та приховування опису платформи, зміна місії організації, зміна фону сторінки, підказки при наведенні на пункти меню та форма для пропозиції нової ініціативи.

Форма перевіряє, чи заповнені всі поля, а також чи містить email символ `@`. Якщо дані некоректні, користувач бачить повідомлення про помилку. Якщо дані введено правильно, пропозиція динамічно додається до сторінки без її перезавантаження.

## Контрольні запитання

### 1. Яку роль відіграє JavaScript у веб-розробці та як він взаємодіє з HTML і CSS?

JavaScript додає веб-сторінкам інтерактивність і динамічну поведінку. HTML відповідає за структуру сторінки, CSS — за її зовнішній вигляд, а JavaScript — за логіку, обробку подій, зміну елементів і реакцію на дії користувача.

Наприклад, HTML створює кнопку, CSS задає їй колір і форму, а JavaScript визначає, що станеться після натискання на цю кнопку.

### 2. Що таке об'єктна модель документа (DOM) і як JavaScript може змінювати її? Наведіть приклад.

DOM — це об’єктна модель HTML-документа, яка представляє сторінку як дерево елементів. JavaScript може звертатися до цих елементів, змінювати їхній текст, стилі, атрибути, додавати або видаляти блоки.

Приклад:

```javascript
const title = document.querySelector('h1');
title.textContent = 'Новий заголовок';
```

У цьому прикладі JavaScript знаходить перший заголовок `h1` і змінює його текст.

### 3. Як працює обробка подій у JavaScript? Наведіть приклад використання addEventListener().

Обробка подій дозволяє виконувати певний код у відповідь на дії користувача: клік, введення тексту, наведення миші, відправлення форми тощо. Метод `addEventListener()` додає слухач події до певного елемента.

Приклад:

```javascript
const button = document.querySelector('#myButton');

button.addEventListener('click', () => {
  alert('Кнопку натиснуто!');
});
```

Коли користувач натискає кнопку, виконується функція і з’являється повідомлення.

### 4. Які є основні типи циклів у JavaScript і коли варто використовувати for, while та forEach?

У JavaScript найчастіше використовують цикли `for`, `while` та `forEach`.

Цикл `for` зручно використовувати, коли відома кількість повторень. Наприклад, коли потрібно пройти по масиву за індексами.

Цикл `while` використовується тоді, коли дія повинна повторюватися, поки виконується певна умова. У цій лабораторній роботі `while` використано для проходження по масиву ініціатив.

Метод `forEach` зручно використовувати для перебору масиву або колекції елементів, коли потрібно виконати дію для кожного елемента.

### 5. Як умовні оператори (if-else) допомагають створювати динамічну поведінку веб-сторінки? Наведіть приклад.

Умовні оператори дозволяють виконувати різні дії залежно від ситуації. Наприклад, можна перевірити, чи заповнена форма, чи користувач уже приєднався до ініціативи, або чи є ще вільні місця для волонтерів.

Приклад:

```javascript
if (volunteersNeeded > 0) {
  volunteersNeeded--;
} else {
  console.log('Команду вже зібрано');
}
```

У цьому прикладі кількість потрібних волонтерів зменшується тільки тоді, коли вона більша за нуль.
