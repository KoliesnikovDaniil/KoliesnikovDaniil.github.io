REST API

Реалізовані маршрути

GET /api/message — перевірка сервера
GET /api/initiatives — список ініціатив
POST /register — реєстрація користувача
POST /login — вхід і отримання JWT
GET /profile — захищений профіль користувача
GET /api/initiatives/:id/ratings — оцінки та середнє значення
POST /api/initiatives/:id/ratings — додавання оцінки
