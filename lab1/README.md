# Лабораторна робота №1

**Предмет:** Вебтехнології та розробка вебзастосувань  
**Варіант:** 9 — веб-сайт платформи для організації волонтерських ініціатив

## Структура проєкту

- `index.html` — семантична HTML5-сторінка.
- `styles.css` — стилі, CSS Grid, адаптивність і медіа-запити.

## Реалізовано

1. Розділи: «Доступні ініціативи», «Мої ініціативи», «Про нас».
2. Картки волонтерських ініціатив з назвою, датою, місцем, кількістю потрібних волонтерів і описом.
3. CSS Grid для відображення карток ініціатив.
4. Семантичні теги: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`.
5. Адаптивність для планшетів і телефонів через два медіа-запити.
6. Вертикальна навігація на малих екранах.
7. Адаптивні зображення через `max-width: 100%`.

## Контрольні запитання

### 1. Що таке семантичний HTML і які його переваги?

Семантичний HTML — це використання тегів, які описують зміст елементів сторінки. Наприклад: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`. Переваги: краща структура документа, доступність для скрінрідерів, зручність підтримки коду та покращення SEO.

### 2. Які основні методи застосування CSS до HTML-документа?

CSS можна застосувати трьома способами: вбудовано через атрибут `style`, внутрішньо через тег `style` у `head`, зовнішньо через окремий файл `.css`. Найкраща практика — зовнішній CSS, бо код легше підтримувати, повторно використовувати та масштабувати.

### 3. Що таке адаптивний веб-дизайн?

Адаптивний веб-дизайн — це підхід, за якого сайт зручно відображається на різних екранах: комп’ютерах, планшетах і телефонах. Основні техніки: медіа-запити, гнучкі сітки, Flexbox, CSS Grid, адаптивні зображення та відносні одиниці вимірювання.

### 4. Як працюють медіа-запити у CSS?

Медіа-запити застосовують певні стилі лише за конкретних умов, наприклад при певній ширині екрана.

```css
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
```

### 5. Чим відрізняється Flexbox від CSS Grid?

Flexbox зручний для одновимірного розташування елементів у рядку або колонці. CSS Grid краще підходить для двовимірних макетів, де потрібно керувати і рядками, і колонками. У цій роботі CSS Grid використано для карток ініціатив.
